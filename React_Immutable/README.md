# React Immutable

This project focuses on learning and implementing immutability in JavaScript using Immutable.js library.

## Learning Objectives

- Understanding Immutable objects: Who, what, when, where, and why?
- Using the Immutable.js library to bring immutability to JavaScript
- Understanding the differences between List and Map
- Using Merge, Concat, and Deep Merging
- Understanding lazy Seq

## Requirements

- All files interpreted/compiled on Ubuntu 18.04 LTS using node 12.x.x and npm 6.x.x
- All files should end with a new line
- All functions must be exported

## Setup

Install NodeJS 12.11.x and npm 6.11.x

```bash
npm install
```

## Tasks

### 0. Converting into an Immutable object using fromJS

Create a function `getImmutableObject` that converts a JavaScript object into an immutable Map using Immutable.js `fromJS` method.
